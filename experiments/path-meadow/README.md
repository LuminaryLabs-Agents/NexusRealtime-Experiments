# Path Meadow

Procedural browser experiment for a central-tree meadow fidelity target.

## Target

```txt
central hero tree
winding path from foreground to tree
low third-person camera
dense grass, wildflowers, rocks, mushrooms
foreground vegetation clusters
path texture, pebbles, ruts, grass edges
tree roots, branch strokes, dappled shadow
foreground frame and 3D depth cues
full WebGL 3D cel-shaded presentation
atomic idempotent descriptor stream packets
different generation ratios for each entity type
distant tree line
golden-hour sky, clouds, haze bands, hills
```
