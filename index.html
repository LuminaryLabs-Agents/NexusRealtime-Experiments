NexusRealtime Experiments

A tiny deployment target for playable NexusRealtime validation apps: runtime-driven, kit-composed, sequence-orchestrated experiments that can be promoted only after they prove themselves.

Runtime

Deterministic ticks, resources, events, components, systems, surfaces, and sequence execution.

Kits

Reusable coded behavior that owns simulation rules while hosts and renderers stay presentation-only.

Experiments

Small playable proofs that validate game feel, kit composition, renderer boundaries, and promotion readiness.

Deployed from main with GitHub Actions and GitHub Pages.
