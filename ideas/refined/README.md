# Refined Ideas

This folder holds ideas that are clearer than raw notes.

A refined idea should have:

- A strong one-line pitch.
- A playable core loop.
- A clear arcade cabinet fit.
- A simple control scheme.
- A known failure state.
- A rough prototype scope.

## Move Here When

Move a raw idea here when someone can understand the game without needing a long explanation.

## Move Out When

Move an idea to `ideas/ready-for-prototype/` when the smallest playable version is clear enough to build.
