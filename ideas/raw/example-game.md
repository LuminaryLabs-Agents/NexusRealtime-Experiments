# Example Game

## One-Line Pitch

A placeholder example showing how a raw Nexus Arcade idea should begin.

## Core Fantasy

The player immediately understands the action and wants one more run.

## Arcade Cabinet Fit

This idea should be short-session, readable from a distance, and playable with joystick plus two buttons.

## Possible Controls

```txt
Joystick: move
Button A: main action
Button B: secondary action
Home: return to launcher
```

## Core Loop

```txt
start run
  ↓
move through danger
  ↓
score points
  ↓
difficulty rises
  ↓
fail or restart
```
