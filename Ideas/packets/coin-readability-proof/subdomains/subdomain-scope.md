# Subdomain Scope

## Possible subdomains

```txt
coin-silhouette
metal-material-response
pickup-highlight-state
distance-readability-lod
small-token-budget-validation
```

## Required first-pass subdomains

```txt
coin-silhouette: stable round token shape
metal-material-response: readable gold/silver/copper variants
pickup-highlight-state: object emphasis without hiding shape
distance-readability-lod: near/mid/far clarity
```

## Rejected subdomains

```txt
coin-economy-rules
coin-specific-renderer
coin-gameplay-scoring
```
