# Concept

## Title

Coin Readability Proof

## One-line pitch

A repeatable coin generator that proves tiny pickup readability, metal material response, pickup glow, and distance-based LOD clarity.

## Object fantasy

A coin that instantly reads as valuable, collectible, and arcade-friendly from near, mid, and far distances.

## Scope

This is a single collectible object proof.

It does not require scoring, inventory, or pickup gameplay yet.

## Smallest useful version

```txt
one coin mesh
six material variants
one glow state
three distance snapshots
one descriptor snapshot
```
