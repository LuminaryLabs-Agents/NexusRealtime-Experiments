# Object Proof

This packet proves a tiny collectible object.

The object is a simple coin-like token.

The same seed should create the same mesh, material, highlight state, distance notes, and budget fields.

## Required outputs

- mesh descriptor
- material descriptor
- highlight descriptor
- LOD descriptor
- readability descriptor
- budget descriptor

## Passes

- simple silhouette
- beveled rim
- material variants
- highlight state
- distance readability checks
