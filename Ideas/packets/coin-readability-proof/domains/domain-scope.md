# Domain Scope

## Possible domains

```txt
pickup-object
material-palette
readability-testing
lod-policy
performance-budget
visual-feedback
```

## Required first-pass domains

```txt
pickup-object: stable token object descriptors
material-palette: metal material variants
readability-testing: near/mid/far review descriptors
performance-budget: triangle and texture budget reports
```

## Domain-update-first audit

Prefer updating existing visual, material, and performance kits before creating new coin-specific kits.

No coin-named DSK should be created.
