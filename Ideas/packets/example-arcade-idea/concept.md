## One-line pitch

A simple placeholder concept used to demonstrate the idea packet format.

## Core fantasy

The player understands the action immediately and wants one more run.

## Arcade / experiment fit

Short session.

Readable from a distance.

Playable with simple input.

Easy to prototype.

## Controls

```txt
Joystick / WASD: move
Button A / Space: primary action
Button B / E: secondary action
R: restart
```

## Core loop

```txt
start run
  ↓
move through danger
  ↓
score points
  ↓
difficulty rises
  ↓
fail or restart
```
