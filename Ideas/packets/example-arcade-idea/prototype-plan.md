# Prototype Plan

## Prototype question

Does the core loop feel understandable and replayable within one minute?

## Smallest playable version

```txt
one arena
one player action
one hazard
one score rule
one failure state
one restart loop
```

## Success criteria

- Understandable in 5 seconds.
- Simple controls.
- Clear failure.
- Player wants another run.
