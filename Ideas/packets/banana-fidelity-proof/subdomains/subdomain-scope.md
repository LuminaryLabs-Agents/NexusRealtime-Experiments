# Subdomain Scope

## Possible subdomains

```txt
organic-mesh-generation
fruit-skin-material-response
blemish-texture-layering
small-object-lod-generation
texture-budget-validation
```

## Required first-pass subdomains

```txt
organic-mesh-generation: stable curved banana silhouette
fruit-skin-material-response: yellow peel material and roughness
blemish-texture-layering: deterministic spots and bruises
small-object-lod-generation: readable near/mid/far versions
```

## Rejected subdomains

```txt
banana-gameplay-rules
banana-inventory-system
banana-specific-renderer
```
