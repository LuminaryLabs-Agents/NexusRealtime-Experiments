# Concept

## Title

Banana Fidelity Proof

## One-line pitch

A repeatable banana object generator that proves organic mesh shape, fruit-skin material response, blemish texture layering, and small-object LOD stability.

## Object fantasy

A simple banana that looks readable, organic, slightly imperfect, and game-ready from near and mid distance.

## Scope

This is a single object proof.

It does not need player movement, inventory, scoring, or a full scene.

## Smallest useful version

```txt
one banana mesh
one peel material
one spot/blemish layer
one normal/roughness response
three LOD levels
one descriptor snapshot
```
