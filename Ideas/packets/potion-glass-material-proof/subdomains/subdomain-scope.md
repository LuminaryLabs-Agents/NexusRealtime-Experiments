# Subdomain Scope

## Possible subdomains

```txt
glass-material-response
liquid-fill-descriptor
bottle-label-texture
emissive-liquid-state
cheap-refraction-fake
small-item-lod
```

## Required first-pass subdomains

```txt
glass-material-response: readable transparent bottle material
liquid-fill-descriptor: fill amount and color descriptors
bottle-label-texture: stable label layer
emissive-liquid-state: glow cue as separate state
small-item-lod: near/mid/far item readability
```

## Rejected subdomains

```txt
potion-gameplay-rules
potion-specific-renderer
inventory-consumption-rules
```
