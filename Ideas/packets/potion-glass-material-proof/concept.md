# Concept

## Title

Potion Glass Material Proof

## One-line pitch

A repeatable potion bottle generator that proves glass material response, liquid fill descriptors, label texture layering, and emissive fantasy item readability.

## Object fantasy

A small bottle that feels magical, readable, and valuable without requiring a full game system.

## Scope

This is a single object proof.

It does not need inventory, crafting, or combat rules.

## Smallest useful version

```txt
one bottle mesh
one glass material
three liquid colors
one label texture
one glow state
one descriptor snapshot
```
