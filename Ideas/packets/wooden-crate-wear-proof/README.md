# Wooden Crate Wear Proof

A small idempotent object proof for generated prop wear, wood texture, and reusable object state descriptors.

## Goal

Generate a crate from a fixed spec and repeatedly improve wood grain, edge wear, wetness, and readable prop-state variants.
