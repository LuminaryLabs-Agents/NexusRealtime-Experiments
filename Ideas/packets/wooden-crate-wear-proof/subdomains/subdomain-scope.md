# Subdomain Scope

## Possible subdomains

```txt
wood-grain-texture
edge-wear-mask
wetness-material-response
box-prop-lod
surface-overlay-budgeting
```

## Required first-pass subdomains

```txt
wood-grain-texture: stable procedural wood texture layer
edge-wear-mask: readable edge use marks
wetness-material-response: roughness and color shift
box-prop-lod: near/mid/far box readability
```

## Rejected subdomains

```txt
crate-gameplay-rules
crate-specific-renderer
inventory-container-rules
```
