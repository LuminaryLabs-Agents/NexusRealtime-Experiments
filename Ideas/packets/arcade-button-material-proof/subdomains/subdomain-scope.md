# Subdomain Scope

## Possible subdomains

```txt
button-press-state
plastic-material-response
led-glow-state
surface-wear-texture
cabinet-control-descriptor
```

## Required first-pass subdomains

```txt
button-press-state: stable visual state transition
plastic-material-response: readable glossy cap material
led-glow-state: lit/unlit descriptor difference
surface-wear-texture: deterministic usage marks
```

## Rejected subdomains

```txt
arcade-button-gameplay-rules
cabinet-specific-renderer
score-from-button-press
```
