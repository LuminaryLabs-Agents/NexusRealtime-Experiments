# Concept

## Title

Arcade Button Material Proof

## One-line pitch

A repeatable arcade button generator that proves physical interactor states, glossy plastic material response, LED glow, and worn surface overlays.

## Object fantasy

A real-feeling cabinet button that visually responds to being pressed, lit, and worn over time.

## Scope

This is a single interactive object proof.

It does not require a full input system or cabinet simulation.

## Smallest useful version

```txt
one button mesh
unpressed state
pressed state
LED-lit state
worn plastic texture
one descriptor snapshot
```
