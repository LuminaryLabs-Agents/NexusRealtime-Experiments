# Object Proof Idea Packets

These five packets track small, idempotent, high-fidelity object proofs.

They are not full games.

They are repeatable object-generation and optimization experiments.

## Packet tree

```txt
Ideas/
└─ packets/
   ├─ banana-fidelity-proof/
   ├─ coin-readability-proof/
   ├─ arcade-button-material-proof/
   ├─ wooden-crate-wear-proof/
   └─ potion-glass-material-proof/
```

## Shared proof goal

```txt
each packet
└─ must prove:
   ├─ same seed produces same object
   ├─ same spec produces same descriptor tree
   ├─ optimization passes are trackable
   ├─ visual variants are comparable
   ├─ performance budget is measurable
   ├─ renderer stays presentation-only
   └─ possible ProtoKit domains are clearly scoped
```

## Packet purpose

- `banana-fidelity-proof` proves organic prop generation and fruit-skin texture optimization.
- `coin-readability-proof` proves tiny pickup readability and metal material response.
- `arcade-button-material-proof` proves physical interactor states and LED/plastic material feedback.
- `wooden-crate-wear-proof` proves simple prop generation, damage states, and wear overlays.
- `potion-glass-material-proof` proves glass, liquid, label, and emissive fantasy item descriptors.
